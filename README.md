# 多附表 Excel 时长自动累加工具

根据姓名自动匹配，支持多个附表累加到主表，自动清空 0 值。

## 功能
- 选择主表
- 添加多个附表
- 每个附表可选择累加到不同列
- 自动清空所有为 0 的单元格
- 不修改原文件，导出新表

## 运行环境与依赖
### 运行环境
Python 3.8+

### 安装依赖
在终端执行以下命令，安装所需的库：
